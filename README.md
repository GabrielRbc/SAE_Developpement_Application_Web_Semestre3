# Développer une application web permettant la gestion des adhérents d’une association.

## Description

Les associations (sportives, culturelles, caritatives, etc.) ont besoin de gérer leurs adhérents :
inscriptions, cotisations, activités, etc.
Aujourd’hui, beaucoup utilisent encore des fichiers Excel. Ce mode de gestion présente plusieurs
inconvénients :
- Fichier difficilement partageable entre membres du bureau,
- Risque de perte ou de corruption du fichier,
- Absence de sécurité ou d’historique des modifications,
- Accès limité à un seul poste ou logiciel spécifique,
- L’objectif de ce projet est de remplacer ces listes Excel par un outil web accessible de partout et sur
n’importe quel appareil afin de pouvoir faire cette gestion.
